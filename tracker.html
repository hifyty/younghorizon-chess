⚙️ First-Time Setup

The tracker needs Firebase for secure login + data storage, and optionally EmailJS for sending parent emails. Follow the steps below, then click Save & Launch.

1Create a Firebase Project

  1. Go to console.firebase.google.com → Add project
  2. Enable Authentication → Sign-in method → Email/Password
  3. Add your admin user under Authentication → Users tab
  4. Enable Firestore Database → Start in Production mode
  5. In Firestore Rules, set: allow read, write: if request.auth != null;
  6. Go to Project Settings → Your apps → ⊕ Web app → copy the config values below

2Firebase Config

3Brevo — for sending parent emails (free)

  1. Go to brevo.com → create a free account (300 emails/day free)
  2. Go to SMTP & API → API Keys → click Generate a new API key → copy it
  3. Go to Senders & IP → Senders → add and verify the email address you'll send from
  4. Paste the API key and your sender email below
